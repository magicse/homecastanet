Usage Guide
How to set up your Chromecast without Google Home
Prerequisites

You need:

▸ Chromecast connected via Ethernet (or on its setup hotspot)

▸ Chromecast's IP address — find it with Wireshark or your router

▸ Target WiFi SSID and password

▸ This tool running on the same network as the Chromecast

Step by Step

1. Enter the Chromecast IP and click Connect to Device

2. Click Scan for Networks — wait ~20s for results

3. Click your target WiFi network in the list

4. Enter the WiFi password and click Connect Chromecast

5. Unplug the Ethernet cable — the Chromecast won't use WiFi while Ethernet is connected!

⚠️ Important Notes

• If Ethernet is still plugged in, the Chromecast will ARP for its WiFi IP on Ethernet and drop the WiFi connection. Unplug it!

• The device uses TLS 1.2 with a self-signed cert — connections are still encrypted

• Password is RSA-encrypted before sending to the device

• keep_hotspot_until_connected: true is sent so setup via hotspot is possible
